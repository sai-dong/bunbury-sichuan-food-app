# Bunbury 健康川菜菜谱库

一个本地静态 App，用来维护健康川菜菜谱库、生成一周计划、按 Spudshed / Woolworths 汇总购物清单，并显示每人每顿和每天的大概热量。

## 打开方式

直接用浏览器打开 `index.html` 即可，不需要安装依赖或启动服务器。

## 功能

- 内置 30 道健康川菜：8 个早餐、22 个中/晚餐。
- 中/晚餐共用同一类菜谱和同样份量，符合日常备餐逻辑。
- 每道菜包含类型、每人热量、蛋白质、食材、做法、标签。
- 一周计划可重新生成，也可以在每天的下拉框里手动换菜。
- 购物清单根据一周计划自动汇总，并按大采购 / 补货拆分。
- 可手动加菜谱，或让 AI 生成同格式 JSON 后粘贴导入。
- 自定义菜谱、设置、勾选状态和备注会保存在浏览器本地。
- Supabase 云同步：你和老婆使用同一个同步码，共享计划、菜谱和购物勾选状态。

## AI 导入格式

```json
[
  {
    "name": "椒麻鸡胸蒸茄子",
    "types": ["main"],
    "calories": 460,
    "protein": 42,
    "tags": ["椒麻", "蒸菜", "高蛋白"],
    "ingredients": [
      { "name": "鸡胸肉", "amount": 160, "unit": "g", "store": "Woolworths" },
      { "name": "茄子", "amount": 1, "unit": "根", "store": "Spudshed" }
    ],
    "steps": ["鸡胸切片腌10分钟", "茄子垫底蒸熟", "出锅淋椒麻汁"]
  }
]
```

## 注意

热量是粗估，用来帮助计划和对照趋势。包装食品请优先看营养标签；如果有慢性病、特殊训练计划或体重变化异常，建议问医生或营养师。

## 云同步

默认状态下，App 仍然可以完全离线使用。要让两个人直接同步数据，请按 `SUPABASE_SETUP.md` 配置 Supabase，然后在 App 里创建同步码并发给另一台设备。

底部不再提供共享 JSON 导入导出；家庭共享走同步码，AI 加菜仍然保留 JSON 导入。
